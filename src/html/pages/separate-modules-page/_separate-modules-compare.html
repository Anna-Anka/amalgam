два типа модулей

Отдельный модуль

Разработан как отдельно стоящий модуль, однако их можно ставить друг на друга (2 этажа), делая отдельный вход. Возможно объединение модулей, с сохранением стен.

Для зданий со свободными планировками

Позволяет возводить сложные модульные здания с любой планировкой до 3-х этажей за счет усиленного каркаса

Предназначен для умеренного климата

Модуль надежный и теплый, для большей части России.

Для суровых условий

Предназначен для эксплуатации в жёстких климатических условиях. Есть возможность увеличить толщину утеплителя по необходимости.

Базовое решение

Подходит для стандартных задач, которые решаются отдельными модулями: расселение рабочих, душ и санузел, пост охраны.

Универсальное решение

Для общежитий, офисных зданий, гостиниц, зданий специального назначения.

  • Стойки — гнутый профиль специального сечения, 3 мм, 120 х 180 мм
  • Основание — гнутый швеллер 100 х 50 х 4
  • Утеплитель (кровля х стены х потолок) — 100 х 100 х 100 мм
  • Кровля — каркас — швеллер гнутый, 140 х 60 х 4
